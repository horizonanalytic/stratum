# Examples

This section contains example programs demonstrating Stratum's features and capabilities. The examples are organized by complexity level:

- **Simple** - Basic programs for beginners learning Stratum
- **Intermediate** - Programs demonstrating common patterns and standard library usage
- **Advanced** - More complex examples showing real-world use cases

## Running Examples

All examples can be run using the Stratum CLI:

```bash
stratum run examples/simple/hello_world.strat
```

Or clone the repository and run from the examples directory:

```bash
git clone https://github.com/cachemcclure/stratum
cd stratum
stratum run examples/simple/hello_world.strat
```
